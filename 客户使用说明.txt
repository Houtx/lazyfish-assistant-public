懒鱼助手 - 客户安装说明
========================

安装前准备
----------
电脑需要先安装并启动 Docker Desktop：
https://www.docker.com/products/docker-desktop/

Windows
-------
1. 解压整个 ZIP 文件，不要只打开压缩包。
2. 双击“安装或更新懒鱼助手.bat”。
3. 首次下载可能需要几分钟，完成后浏览器会自动打开。
4. 在页面输入卖家提供的授权码。

macOS
-----
1. 解压整个 ZIP 文件。
2. 首次运行时，右键点击“安装或更新懒鱼助手.command”，选择“打开”。
3. 系统询问时确认打开；完成后浏览器会自动打开。
4. 在页面输入卖家提供的授权码。

日常使用
--------
- 安装和更新：双击“安装或更新懒鱼助手”。
- 只启动、不更新：双击“启动懒鱼助手”。
- 临时停止：双击“停止懒鱼助手”。
- 访问地址：http://127.0.0.1:9000

如果端口 9000 已被其他软件占用，首次安装会自动选择 9001-9099 中的可用端口，实际地址会显示在安装窗口中。

重要提醒
--------
- 不要删除或修改 .env、docker-compose.yml 和 global_config.yml。
- 不要执行 docker compose down -v，这会删除业务数据。
- 更新不会删除数据库、Cookie、浏览器状态、上传文件或日志。
- 迁移电脑或重装系统前，请先联系卖家协助备份数据。
