懒鱼助手 - 客户安装说明
========================

产品官网
--------
https://lazyfish.haoyunqiankun.com/

官网提供产品功能、数据与授权说明、安装流程和官方下载入口。授权购买入口开放后，以官网展示的信息和卖家正式上架渠道为准。

安装前准备
----------
电脑需要先安装并启动 Docker Desktop：
https://www.docker.com/products/docker-desktop/

Windows
-------
1. 解压整个 ZIP 文件，不要只打开压缩包。
2. 双击“安装或更新懒鱼助手.bat”。
3. 首次下载可能需要几分钟，完成后浏览器会自动打开。
4. 在页面输入卖家提供的授权码。
5. 激活后使用登录页显示的 admin 初始密码登录，并按提示设置新密码。
6. 安装窗口会显示 noVNC 人工验证入口和随机密码；需要处理滑块时按后台提示使用。

macOS
-----
1. 解压整个 ZIP 文件。
2. 首次运行时，右键点击“安装或更新懒鱼助手.command”，选择“打开”。
3. 系统询问时确认打开；完成后浏览器会自动打开。
4. 在页面输入卖家提供的授权码。
5. 激活后使用登录页显示的 admin 初始密码登录，并按提示设置新密码。
6. 安装窗口会显示 noVNC 人工验证入口和随机密码；需要处理滑块时按后台提示使用。

Linux 云服务器
--------------
1. 使用 SSH 登录云服务器。
2. 复制下面一整行命令并执行，不需要下载或解压安装包：

   curl -fsSL https://lazyfish.haoyunqiankun.com/install.sh | sudo bash

3. 脚本会自动安装 Docker、选择 9000-9099 中的可用端口、拉取最新稳定版并启动。
4. 根据脚本最后显示的端口，在云厂商安全组中放行对应的 TCP 端口。
5. 打开脚本显示的公网地址，输入卖家提供的授权码完成激活。
6. 需要人工处理滑块时，按安装器显示的命令建立 SSH 隧道，再使用 noVNC 入口和密码。不要在安全组中开放 5900 或 6080。

以后再次执行同一句命令就是更新，不会删除授权、配置或业务数据。

Linux 常用命令
-------------
- 安装或更新：sudo lazyfish-assistant update
- 启动：sudo lazyfish-assistant start
- 停止：sudo lazyfish-assistant stop
- 查看状态：sudo lazyfish-assistant status
- 查看日志：sudo lazyfish-assistant logs

日常使用
--------
- 安装和更新：双击“安装或更新懒鱼助手”。
- 只启动、不更新：双击“启动懒鱼助手”。
- 临时停止：双击“停止懒鱼助手”。
- Windows/macOS 访问地址：http://127.0.0.1:9000
- Linux 访问地址：以安装脚本最后显示的公网地址为准
- noVNC 人工验证：以安装窗口或后台显示的入口和密码为准；必须在容器浏览器中完成滑块

如果端口 9000 已被其他软件占用，首次安装会自动选择 9001-9099 中的可用端口，实际地址会显示在安装窗口中。

重要提醒
--------
- 不要删除或修改 .env、docker-compose.yml、docker-compose.vnc.yml、global_config.yml 或 secrets/vnc_password.txt。
- 不要执行 docker compose down -v，这会删除业务数据。
- 更新不会删除数据库、Cookie、浏览器状态、上传文件或日志。
- 迁移电脑或重装系统前，请先联系卖家协助备份数据。
- 不要向第三方提供授权码、Cookie、管理员密码或业务数据。
- 忘记管理员密码时，可在登录页使用当前安装绑定的有效授权码重置。
- 底层 VNC 5900 不发布到宿主机，noVNC 6080 只允许绑定 127.0.0.1；不要向公网开放这些端口。
